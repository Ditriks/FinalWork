**Условие задачи:**

*Написать программу, которая из имеющегося массива строк формирует массивиз строк, длина которых меньше либо равна 3.*

**Алгоритм решения задачи:**

1.Вводим через запятую элементы массива

2.Проверяем циклом каждый элемент на соответствие условию - длина строки меньше или равна трем.

3.Если условие выполняется - записываем элемент в новый массив.

4.Выводим результат - новый массив.

*Блок-схема решения и код приложены отдельными файлами.*
